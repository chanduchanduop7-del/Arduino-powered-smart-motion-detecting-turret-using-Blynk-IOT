# Arduino-Powered Smart Motion Detecting Turret Using Blynk IoT

## Overview
This project is an IoT-based smart motion-detecting turret developed using Arduino and the Blynk IoT platform. The system automatically detects moving objects using a motion sensor and tracks them by rotating servo motors. Users can also monitor and control the turret remotely through the Blynk mobile application over Wi-Fi.

## Features
- Automatic motion detection
- Real-time object tracking
- 2-axis servo motor control
- Remote monitoring using Blynk IoT
- Manual turret control through a smartphone
- Wi-Fi connectivity
- Low-cost embedded system design

## Hardware Components
- Arduino Uno/Nano
- ESP8266 Wi-Fi Module (or ESP32)
- HC-SR04 Ultrasonic Sensor / PIR Motion Sensor
- 2 × Servo Motors (SG90/MG996R)
- Jumper Wires
- Breadboard
- Power Supply

## Software Used
- Arduino IDE
- Embedded C
- Blynk IoT
- Servo Library

## Working
The motion sensor continuously monitors the surrounding area. When an object is detected, the Arduino processes the sensor data and rotates the servo motors to track the object's movement. The ESP8266/ESP32 sends the sensor status to the Blynk cloud, allowing users to monitor and manually control the turret using the Blynk mobile application.
