# Arduino-Powered Smart Motion Detecting Turret Using Blynk IoT

## 📌 Overview
The Arduino-Powered Smart Motion Tracking Turret is an embedded systems project that automatically detects and tracks nearby objects using an ultrasonic sensor mounted on a servo motor. The system continuously scans its surroundings, measures object distance, and displays the detected objects on a radar-style interface. It also provides alerts through a buzzer and sends notifications using an ESP32 with Blynk IoT.

---

## 🎯 Objectives

- Scan the surrounding area using a servo-mounted ultrasonic sensor.
- Detect objects by measuring their distance at different angles.
- Process the collected data using Arduino Uno.
- Display object positions on a real-time radar interface.
- Send notifications through ESP32 and Blynk IoT.
- Trigger a buzzer when an object enters the predefined detection range.

## ⚙️ Working Principle

1. The servo motor rotates the ultrasonic sensor across a predefined angle.
2. The ultrasonic sensor measures the distance of nearby objects.
3. Arduino Uno processes the distance and angle data.
4. Processing 3 displays the data as a radar-like visualization.
5. ESP32 sends notifications through the Blynk IoT platform.
6. A buzzer alerts the user when an object is detected within the threshold distance.
